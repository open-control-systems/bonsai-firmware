## Installation Instructions

```bash
npm create vite@latest web-gui -- --template preact
cd web-gui
npm install --save-dev typescript vite-plugin-compression eslint eslint-plugin-prettier
npm install
```

## Development

```bash
# Build the project
npm run build

# Run local HTTP server
npm run dev

# Format the source code
npx eslint --fix .
```
